# CapooPals

### How to play?

扣B，在控制台输入咖波序号（1~126）可以添加新的咖波

摁C，可以清除所有咖波

敲Q，退出程序

拖动或抚摸咖波来与其互动吧。

### How to compile?

你需要cmake和c++编译器

```
mkdir build && cd build;
cmake ../;
cmake --build .;
```
